# Rizky Widyayulianto — Resume

This is a personal resume website built with **Next.js** and **Tailwind CSS**.

> ⚠️ **Disclaimer:** This project was created with the assistance of **DeepSeek (via GitHub Copilot)** and is deployed on **Netlify**.

## Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS v4
- **Deployment:** Netlify

## Getting Started

```bash
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Deployment

The site is configured for **Netlify** via `netlify.toml`. Push to the `main` branch and Netlify will automatically build and deploy.
